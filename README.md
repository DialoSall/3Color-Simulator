# 3Color

3Color is an interactive browser puzzle game based on the graph 3-coloring problem.

Live site: https://3color.vercel.app/

## Features

- 15 progressive campaign levels
- Random graph mode
- Conflict detection
- Recolor, reset, and timer tracking
- Mobile-friendly layout
- Social sharing preview and analytics

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Vercel

## Background

This project grew out of an earlier 3-coloring algorithm project, where I explored approaches to graph coloring. I built this version to make the concept more visual and accessible to players without a computer science background.
